AWS Lambda EC2 Cost Guardian

EventBridge · Lambda · SNS · Terraform · Python

  • Built a fully serverless pipeline that automatically monitors running EC2 instances daily and sends detailed email alerts via SNS — solving the real problem of forgotten instances causing unexpected cloud bills.
  • Developed core Lambda function in Python using boto3 to call ec2:DescribeInstances, extracting instance ID, name, type, region, AZ, running duration, and IP addresses.

  • Implemented IAM least privilege: Lambda role limited to only ec2:DescribeInstances and sns:Publish — zero hardcoded credentials.
